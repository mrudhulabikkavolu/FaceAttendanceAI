# 🤖 Face Attendance AI

An AI-powered attendance management system that automatically detects and recognizes faces using computer vision techniques to mark attendance in real time.

## ✨ Features

- 🎥 Real-time face detection
- 👤 Facial recognition-based attendance
- 📝 Automatic attendance recording
- 🖥️ Desktop GUI built with Tkinter
- ⚡ Fast and contactless attendance
- 📅 Date & time stamped attendance logs

---

## 🛠 Tech Stack

- Python
- OpenCV
- face_recognition
- Tkinter
- NumPy
- Pandas

## ⚙️ Installation

```bash
git clone https://github.com/mrudhulabikkavolu/FaceAttendanceAI.git
cd FaceAttendanceAI
pip install -r requirements.txt
python gui.py
```

---

## 🔄 Workflow

1. Register users
2. Capture face images
3. Train the recognition model
4. Detect faces in real time
5. Automatically mark attendance

---

## 🔒 Privacy

The face image dataset is intentionally excluded from this repository to protect user privacy.

Users can create their own dataset before training the recognition model.
